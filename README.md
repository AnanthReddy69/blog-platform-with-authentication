# Blog Platform with Authentication

A full-stack web application that allows users to register, log in, and manage blog posts. The platform includes secure user authentication using bcrypt and sessions, CRUD operations for blog posts, and MongoDB Atlas integration for cloud database storage.

## Features
- User Registration
- User Login & Authentication
- Create Blog Posts
- View Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- MongoDB Atlas Database
- Session Management
- Responsive EJS Views

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- EJS
- Express Session
- Bcrypt.js
